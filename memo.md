# 開発メモ

## プロジェクト作成
```bash
npx create-expo-app@latest tsumori-pay --template blank-typescript
```

## スタート
```bash
npx expo start
```

必要なライブラリのインストール

```bash
npm install expo-router react-native-safe-area-context react-native-screens expo-linking expo-constants expo-status-bar react-native-web react-dom
```

```bash
npx expo install --fix

npx expo customize tsconfig.json
```
